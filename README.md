# Elm MIDI

Fork of [elm-comidi](https://github.com/newlandsvalley/elm-comidi).

## Install & Usage

1. `npm i pravdomil/elm-midi --save-dev`
1. Add `node_modules/elm-midi/src` into `elm.json`/`source-directories`.
1. Use `Midi` module.
